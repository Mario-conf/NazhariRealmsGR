# **App Name**: Altavia Trail

## Core Features:

- Advanced Route Search: Search for hiking routes based on various parameters like difficulty, duration, terrain, date, and presence of historical elements.
- Route Display: Displays hiking trails as clear and informative cards in an Airbnb-style grid.
- Favorite Trails: Users can save favorite trails with local persistence.
- Route Sorting: Sort the listed routes based on criteria like distance, duration, or difficulty level to easily find trails that best match current priorities. Options include name, distance, duration, difficulty, and user ratings.
- Route Reviews: User can give a trail a rating from 1 to 5 and include a text-based review, to help others assess the experience. These user ratings are available for other users to view and take into consideration.

## Style Guidelines:

- Primary color: #2E4F2E to evoke nature and adventure.
- Background color: #C2B280 for a clean, earthy feel.
- Accent color: #5C4033 to highlight key actions and elements.
- Body text font: 'PT Sans', a humanist sans-serif.
- Headline font: 'Playfair', a modern serif. If longer text is anticipated, use 'PT Sans' for body text.
- Use outdoor-themed icons for categories and filters.
- Modern layout inspired by Airbnb for easy navigation.
